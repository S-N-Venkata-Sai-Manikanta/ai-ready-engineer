Course Structure
Full Syllabus

24 hours across 8 sessions — from ReAct loops to production multi-agent teams

Session 1
What Are AI Agents? The Agentic Paradigm
Agent architecture overview — perception, reasoning, tool use and action. How agents differ from simple chatbots. Real-world agent examples.
2 Hours
Session 2
ReAct Pattern — Reasoning + Acting
Implement the ReAct loop from scratch. Build an agent that reasons step-by-step and takes actions based on observations.
3 Hours
Session 3
Tool Use & Function Calling in Agents
Give agents access to web search, code execution, file read/write and API calls. The agent decides which tool to use and when.
3 Hours
Session 4
AI Memory — Short, Long-Term & Episodic
In-context memory, external memory stores, episodic recall with vector databases. Building agents that remember across sessions.
3 Hours
Session 5
LangChain Agents & Chains
Build production agents with LangChain — tools, agents, chains, callbacks and observability. Deploy a LangChain agent with FastAPI.
4 Hours
Session 6
Multi-Agent Systems with CrewAI
Design teams of specialized agents — researcher, writer, reviewer, coder. CrewAI's role, task, and crew abstractions.
4 Hours

Model Context Protocol (MCP) Basics
Anthropic's open standard for connecting AI models to tools and data. Build an MCP server and connect it to Claude agents.

For Trainers
Delivery Guide — How to Teach This Module

Recommended teaching flow, classroom techniques and assessment approach for trainers.

01 · Before Class
Pre-Class Preparation
  • Confirm Module 6 (AI API Integration) is complete — agents require function-calling fluency
  • Pre-install LangChain, CrewAI, LangSmith and a sandboxed code-execution tool
  • Prepare an MCP server demo connected to Claude before Session 7
  • Set up a dedicated OpenAI / Anthropic billing project — agents burn tokens fast
02 · In Class
Live Delivery Style
  • Build the ReAct loop from scratch on screen — no frameworks — before showing LangChain
  • Stream the agent's reasoning trace live so learners see "thinking → acting → observing"
  • Show at least one agent failure mode live and walk through fixing it
  • Compare LangChain vs CrewAI side-by-side on the same problem
03 · Engagement
Student Engagement Tactics
  • Run an "Agent Olympics" — each learner submits an agent, peers grade on autonomy and accuracy
  • Reward the most creative tool integration each session
  • Hold "failure show & tell" — share weird and unexpected agent behaviour
  • Encourage learners to demo agents running live during evaluation, not via recordings
04 · Assessment
Assessment Approach
  • Grade on agent autonomy — how few human nudges are needed to complete the task
  • Evaluate the multi-agent project on role design, hand-offs and final output quality
  • Score safety implementation — human-in-loop, sandboxing, cost guards, retries
  • Run a 3-hour timed challenge: build a working agent for a domain learners have never seen
