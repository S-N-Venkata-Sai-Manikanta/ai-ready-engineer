Module Overview
Why This Module Matters
Why You Need This
  • Everything in AI starts with how you communicate — prompting is the universal skill
  • Poor prompts lead to mediocre outputs even from the most powerful models
  • Systematic prompting unlocks productivity gains of 10x in any knowledge work
  • Prompt engineering is the foundation for all advanced AI integration work
  • Employers rank prompting ability as a top requirement for AI roles
Learning Outcomes
  • Write professional-grade prompts for any use case
  • Build reusable prompt templates and libraries
  • Understand how LLMs process and respond to instructions
  • Apply chain-of-thought and structured output techniques
  • Design multi-step AI workflows using prompts alone
  • Optimize prompts for accuracy, consistency and tone
Course Structure
Full Syllabus

18 hours of structured learning across 7 progressive sessions

Session 1
How LLMs Work — Mental Models for Prompting
Understand tokens, context windows, temperature and how models predict text. Build the mental model needed to write effective prompts.
2 Hours
Session 2
Zero-Shot & Direct Prompting
Learn to give clear, direct instructions. Practice the COSTAR and RTF frameworks for structured prompts.
2 Hours
Session 3
Few-Shot & Chain-of-Thought Prompting
Use examples to guide model behaviour. Apply chain-of-thought for complex reasoning tasks and multi-step problems.
3 Hours
Session 4
Role-Based & System Prompting
Assign personas, set system instructions and control tone. Build custom AI assistants with defined personalities and constraints.
3 Hours
Session 5
Structured Output & JSON Prompting
Extract structured data, tables, JSON and formatted outputs reliably. Essential for AI-to-code integration.
2 Hours
Session 6
Prompt Optimization & Iteration Techniques
Diagnose bad prompts and fix them. Use A/B testing, refinement loops and evaluation metrics to improve output quality.
3 Hours
Session 7
Building Prompt Workflows & Templates
Design end-to-end prompt pipelines for real tasks. Build a reusable prompt template library for your domain.
3 Hours
Topics Covered
Core Topics
🎯
Zero-Shot Prompting
Direct instruction without examples. Master clarity, specificity and framing for reliable outputs.
📚
Few-Shot Prompting
Use 2–5 examples to show the model exactly what format and quality you expect.
🧠
Chain-of-Thought
Guide models through multi-step reasoning with structured thinking prompts.
🎭
Role Prompting
Assign expert personas to unlock domain-specific knowledge and communication styles.
🔧
Structured Prompting
COSTAR, RTF and XML-style frameworks for consistent, professional prompt architecture.
⚡
Prompt Optimization
Iterative refinement, A/B comparison and systematic debugging of underperforming prompts.
Hands-On Learning
Practical Activities
Activity 01
Prompt Comparison Challenge
Write 3 versions of the same prompt and compare outputs across ChatGPT, Claude and Gemini. Analyse differences and document winning patterns.
Activity 02
Role Persona Workshop
Build 5 different role-based system prompts for a virtual assistant. Test tone, accuracy and boundaries across each persona.
Activity 03
Structured Output Extraction
Extract structured JSON data from unstructured paragraphs using prompts alone. Build a reusable extraction template.
Activity 04
Prompt Debugging Lab
Fix 10 broken prompts with common failure patterns: ambiguity, missing context, hallucination triggers, and format issues.
